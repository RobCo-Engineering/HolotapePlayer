# Holotape Player

A quick proof of concept for playing HoloTape games using Ruffle.

## Files

For obvious reasons this repo is missing a lot of files.

### SWF's

There are two locations for SWFs to live `/public/HoloTapes/` and `/public/HolotapesApp/`, the former is SWF's from the game files, the latter is SWF's from the companion app, they do differ.

### WAV's

All the sound files to be registered live in `/public/Sounds/`.

## Running

Install all deps first with `npm i` and then:

```
npm run dev
```
